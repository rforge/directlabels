directlabels - utility.function - Positioning Function - draw.rects

Positioning Function that draws boxes around label positions. Need to have previously called calc.boxes. Does not edit the data frame.

draw.rects <- function(d,...){
  hjust <- vjust <- 0.5
  with(d,grid.rect(x,y,w,h,hjust=hjust,vjust=vjust,
                   default.units="native",gp=gpar(col="grey")))
  d
}
